Convert file wmf thành ảnh

website

Để chuyển đổi file wmf thì cần qua 2 lần convert sau:
- Convert file thành pdf
- Convert pdf thành png

unoconv -f pdf -o tmp/temp.pdf image1.wmf
convert -trim tmp/temp.pdf -colorspace RGB tmp/temp.png

Ở đây mình sử dụng unoconv, ngoài ra có thể sử dụng libreoffice. Để convert pdf thành png thì sử dụng ImageMagick
